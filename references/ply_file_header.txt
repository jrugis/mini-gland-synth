

---------------------------------------------------
Mini-gland mesh: ply file header
---------------------------------------------------

---------
ply
format binary_little_endian 1.0
comment Mini-Gland mesh format 1.0
element vertex NV
property float x
property float y
property float z
element face NF
property int v1
property int v2
property int v3
property int face_type
element tetrahedron NT
property int v1
property int v2
property int v3
property int v4
element cell NC
property int cell_type
property list int int face_index
property list int int tetrahedron_index
element duct_vertex NDV
property float x
property float y
property float z
element duct_segment NDS
property int vertex_in
property int vertex_out
property float radius
end_header
---------

where: 
NV = total number of vertices
NF = total number of surface triangle faces
NT = total number of tetrahedra
NC = total number of cells

face_type 0 = apical
          1 = basolateral
          2 = basal

cell_type 0 = acinus
          1 = intercalated duct
	  2 = striated duct		  

note:
- only the apical cells need to contain tetrahedra
